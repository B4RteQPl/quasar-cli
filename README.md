## Installation

To install quasar-cli from npm, run:

```
$ npm install -g quasar-cli
```

## Usage

Some examples:
```
$ quasar create:project &lt;name&gt;
$ quasar release:recommend
$ quasar release:version &lt;version&gt;
```
